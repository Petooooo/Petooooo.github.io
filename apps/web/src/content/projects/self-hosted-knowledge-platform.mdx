## Why This Exists

The platform is designed to be evidence of ownership: code, content, deployment, operations, and automation all live in one version-controlled system.

## Architecture

- Astro builds the static site.
- GitHub remains the source of truth.
- Decap CMS writes Markdown and MDX content back to the repository.
- Docker Compose runs the site on the Raspberry Pi 4.
- Caddy serves the built output and can request HTTPS certificates once the DDNS domain is configured.

## Operational Posture

The first milestone intentionally avoids a database and server-side application runtime. The result is easier to back up, easier to redeploy, and friendlier to Raspberry Pi hardware.
