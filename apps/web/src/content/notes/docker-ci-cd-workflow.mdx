## Build Strategy

For static platforms, a multi-stage Docker build keeps the runtime image small while preserving a repeatable build process.

```bash
docker compose build
docker compose up -d
```

## Deployment Principle

The Raspberry Pi should be able to rebuild from Git with no hidden local state besides environment variables and mounted backups.
