## Routine Deployment

Connect to the server and run the deploy script from the repository root:

```bash
ssh rp4
cd ~/portfolio-platform
./infra/scripts/deploy.sh
```

## Checks

- Confirm the container is healthy.
- Confirm Caddy is listening on ports `80` and `443`.
- Confirm DDNS points to the home network.
- Confirm router port forwarding targets the Raspberry Pi.

## Rollback

Use Git history as the rollback mechanism. Check out a known-good commit and rerun the deploy script.
