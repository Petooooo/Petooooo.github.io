## Routine Deployment

The exact hostnames and paths are intentionally omitted from the public note. The intended shape is:

```bash
ssh <rp4-host>
cd <repo-path>
./infra/scripts/deploy.sh
```

## Checks

- Confirm the repository pulled the expected commit.
- Confirm the Astro build completed.
- Confirm the container or Caddy service restarted cleanly.
- Confirm the public site serves the new build.
- Keep DDNS and router details out of public notes unless intentionally sanitized.

## Rollback

Use Git history as the rollback mechanism. Check out a known-good commit and rerun the deploy script.
