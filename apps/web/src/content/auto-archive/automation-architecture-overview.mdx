## Publication Model

Generated content is useful only when the workflow preserves trust. This archive is designed around a draft-first model:

1. Collect source data.
2. Summarize and classify.
3. Generate an MDX draft.
4. Commit to GitHub.
5. Review in Decap CMS or an editor.
6. Publish by setting `draft: false`.

## Review Checklist

- Confirm the source material is real.
- Remove private data.
- Check commands and dates.
- Add context that automation could not infer.
- Publish only when the note is worth keeping.
