# Content Workflow

## Manual Writing

Use `/admin` for Decap CMS or edit MDX files directly under `apps/web/src/content`.

## Collections

- Projects: portfolio case studies.
- Notes: general technical notes.
- DevOps: operation logs and runbooks.
- Research: math, computation, and study notes.
- Auto Archive: reviewed generated posts.

## Drafts

Drafts use:

```yaml
draft: true
```

Drafts are excluded from public routes.

## Generated Draft Metadata

Generated content must stay unpublished until reviewed:

```yaml
draft: true
autoGenerated: true
generatedAt: "2026-05-13"
reviewed: false
source: "source system or workflow"
sourceUrl: "https://example.com/source"
```
