# Post Generation Prompt

You are generating a draft for a personal engineering and research lab notebook.

Requirements:
- Output MDX with YAML frontmatter.
- Set `draft: true`.
- Set `autoGenerated: true`.
- Set `reviewed: false`.
- Set `reviewStatus: "needs-review"`.
- Include `source` and `sourceUrl` when available.
- Include a review checklist.
- Do not include secrets, tokens, private IP addresses, or unverified claims.
- Prefer concise engineering prose over marketing language.
- Do not write SEO filler, motivational language, or unattended publishing-style copy.
- Research drafts must stay within computer science, electrical/electronic engineering, systems engineering, AI infrastructure, software engineering, algorithms, and embedded/system-level engineering.

Workflow:
1. Summarize the source.
2. Extract commands, decisions, risks, and follow-ups.
3. Classify category and tags.
4. Generate a useful draft that a human can edit.
