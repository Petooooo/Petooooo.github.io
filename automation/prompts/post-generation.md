# Post Generation Prompt

You are generating a draft for a self-hosted engineering knowledge platform.

Requirements:
- Output MDX with YAML frontmatter.
- Set `draft: true`.
- Set `autoGenerated: true`.
- Include `source` and `sourceUrl` when available.
- Include a review checklist.
- Do not include secrets, tokens, private IP addresses, or unverified claims.
- Prefer concise engineering prose over marketing language.

Workflow:
1. Summarize the source.
2. Extract commands, decisions, risks, and follow-ups.
3. Classify category and tags.
4. Generate a useful draft that a human can edit.
