# Raw Note Intake

This folder is for temporary source material that should be turned into reviewed MDX drafts.

Use it for:

- troubleshooting notes
- terminal logs
- deployment notes
- implementation scratch notes
- algorithm study notes
- system design notes
- infrastructure decisions
- debugging notes
- experiment notes
- paper review notes in computer science, electrical/electronic engineering, systems engineering, AI infrastructure, software engineering, algorithms, or embedded/system-level engineering

Do not store secrets here. Redact tokens, private IPs, internal hostnames, usernames, customer data, and private logs before committing.

Suggested subfolders:

- `automation/raw/inbox`: notes waiting to be converted
- `automation/raw/processed`: notes already converted into draft MDX
- `automation/raw/private`: local-only scratch material; keep sensitive files out of Git
